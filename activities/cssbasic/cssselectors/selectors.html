List out the different ways to select an item.

There are a few ways to select items:
- Name
- ID
- Class

What is the difference between an ID and a Class?

ID is a type of selector. It cannot reuse and given name should be a noun.

A Class also a type of selector. It can reuse as many as we want and the given name should be adjective.

What is the difference between a SPAN tag and a DIV tag?

- Span is used to select a portion of the text that we want to edit, but not the whole text.

- Div is used to select the whold text when we want to edit the text.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

header and footer usually refer to the header of the page and the footer of the page, respectively. However div can refer to any general section of a page.

What are different ways to select a color?

Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.
