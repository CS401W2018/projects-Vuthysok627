What is the difference between block tags and inline tags.

A "block tag" starts on a new line and takes up the full width available, while an "inline tag" displays within the same line as surrounding text, only taking up the space needed for its content; essentially, block elements create separate blocks of content, whereas inline elements flow naturally within a line of text.

What is the margin, border, and padding?

Margin : A space around the boader. It is used to distance the element or border.


Border: A line that can be invisible oor visible around the border. It can be also have different color and weight.


Padding: the space between the content and the border of an element.

What are units of measurement you can use in CSS?

Absolute Units: px, pt, PC, in, cm, mm. Relative Units: em, rem, %, ex, ch, fr.

When would we use a figure tag?

We should use the HTML figure tag when we have content that's related to the main content of a document, but can be separated out and still make sense on its own: A graph, A diagram, A photo, A video clip, and A code listing.
