What is the difference between a block and an inline tag? Give examples of each one.

A "block" tag in HTML takes up the full width available on a line and starts on a new line, while an "inline" tag only takes up the space needed by its content and appears on the same line as surrounding text; examples of block tags include p (paragraph) and div, while inline tags include span and a (anchor link).

What is normal flow in HTML?

Normal Flow, or Flow Layout, is the way that Block and Inline elements are displayed on a page before any changes are made to their layout.

What does float do? Why do you need to clear the float?

In CSS, "float" allows an element to be positioned on the left or right side of its container, enabling text to wrap around it.however, because floated elements are removed from the normal document flow,we need to "clear the float" to ensure that subsequent elements appear below the floated element and not alongside it, maintaining proper layout, especially when dealing with elements like footers that should sit below the entire content area.

What do absolute, fixed, and relative do?

In CSS, absolute, fixed, and relative are positioning methods that affect the placement of elements on a web page

How can you make nice navigation buttons for mobile devices?

To create "nice" navigation buttons for mobile devices, focus on making them easy to tap, visually clear, appropriately sized, use intuitive icons when necessary, and maintain consistency across your design; prioritize readability with large enough text, appropriate color contrast, and consider using standard design patterns familiar to mobile users.
