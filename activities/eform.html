The form tag has an "action" attribute. What does it do?

The action attribute defines the URL where the form data will be sent for processing when the form is submitted.

The form tag has a "method" attribute. What does it do?

The method attribute determines how the form data will be sent to the server. The "POST" method sends data securely, while "GET" sends data in the URL.

What is the difference between "Post" and "Get"?

POST sends data in the body of the request, which makes it more secure, especially for sensitive data like passwords. GET appends data to the URL, making it visible in the browser's address bar.
