1. Why is the viewport meta tag important?

It controls how the webpage displays on different devices, especially mobile phones. We can automatically resize the webpage based on the meta tag.

How can you adjust the CSS for printing?

By using the CSS @media at-rule.

When would you want to adjust the CSS for printing?

Depends on how we want our page to look and the size we want.

Why do we need a viewport meta tag?

Because it helps adjust the size of the display on the device, especially mobile phones.

When you select the same tag twice, and one says "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The last tag will take priority.

How do you change the CSS for a web page displayed on a phone?

By using @media queries to set the size of the display.
